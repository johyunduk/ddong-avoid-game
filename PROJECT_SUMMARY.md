# 똥 피하기 게임 - 프로젝트 요약

## 프로젝트 개요

## 실행 방법

### 개발 모드
```bash
npm run dev
```
→ http://localhost:5173 접속

### 프로덕션 빌드
```bash
npm run build
```
→ `dist/` 폴더에 결과물 생성

### 빌드 미리보기
```bash
npm run preview
```

## 게임 조작법

### PC
- **좌우 화살표 키** (← →): 플레이어 이동

### 모바일
- **화면 터치**: 터치한 방향으로 이동
- **게임 오버 후 터치**: 재시작

## 게임 특징

### 난이도 증가 시스템
- 10초마다 난이도 자동 증가
- 똥 낙하 속도 증가
- 똥 생성 주기 단축

### 점수 시스템
- 0.1초마다 1점 증가
- 생존 시간이 곧 점수
- 게임 오버 시 최종 점수 표시

## 토스 SDK 연동 가이드

### 현재 상태 (개발 환경)
- Mock SDK 사용 중
- 콘솔 로그로 동작 확인 가능
- 로컬스토리지에 점수 임시 저장

### 실제 배포 시
1. 토스 SDK 설치
   ```bash
   npm install @toss/games-sdk
   ```

2. 앱 ID 설정
   - 토스 개발자 콘솔에서 앱 등록
   - `src/main.ts`에서 앱 ID 설정

3. SDK 활성화
   - `src/utils/tossSDK.ts`에서 주석 해제
   - Mock SDK 대신 실제 SDK 사용
